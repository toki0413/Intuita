01 / Creative Introduction

【生活娱乐赛道】Intuita——构造即证明

一款 3D 物理构造解谜游戏。玩家在守恒矩阵约束下构建真实材料结构，每一步操作对应一次数学证明。不可判定性被具象化为迷雾，哥德尔不完备定理成为可玩的博弈。LLM 算力由玩家自带 API Key 实现，游戏本身零服务器成本。

⏰
为什么是现在？ LLM API 开放使 BYOK 模式成为可能（2023年前不可行）；Godot 4 的 GDExtension 使 Rust 原生计算可行（Godot 3 不可行）；Z3 SMT Solver 开源使轻量形式化验证可行（Coq/Lean4 太重）。三个技术条件同时成熟，窗口期就在当下。
02 / The Problem

我们发现的核心痛点

科学教育游戏存在一个根本性的断层：要么太"教育"（本质是刷题模拟器），要么太"游戏"（科学只是装饰皮肤）。没有人把数学哲学中"存在即被构造"的直觉主义思想做成可玩的机制。

01

科学游戏的两极分化

现有科学游戏要么是披着科学皮的普通解谜（Minecraft 化学只是换皮合成），要么是毫无游戏性的专业工具（VASP、Gaussian 的学习曲线比游戏本身还陡）。中间地带几乎空白。

02

抽象概念无法"亲手触摸"

晶体学中的空间群、Wyckoff 位置、对称性破缺——这些概念在课本上是二维投影+死记硬背。学生无法直观感受"为什么 NaCl 是 Fm-3m 而不是 Pm-3m"，因为没有人让他们在 3D 空间中亲手构建并观察守恒律的约束。

03

数学哲学被锁在论文里

Brouwer 的直觉主义、哥德尔的不完备定理、范畴论的态射系统——这些概念通常只出现在大学数学系，大多数人没有途径直观地理解它们。"存在即被构造"是一种有趣的世界观——但目前只有少数专业人士接触过它。

04

硬核解谜的深度天花板

Zachtronics 系列证明了解谜游戏可以有真正的工程深度，但通关后玩家找不到更深的挑战。现有解谜游戏的物理规则是简化的、装饰性的，不是真实的科学约束。渴望"规则即物理"的硬核体验的玩家无处可去。

04 / Core Mechanics

四大核心机制

Intuita 的每一个游戏机制都映射一个数学或物理概念。守恒矩阵是真实的线性代数，证明树是真实的构造性逻辑，迷雾是真实的可判定性层次。

Mechanic I — Conservation Matrix

守恒矩阵 — 结构的生死线

一个 4×4 实对称矩阵追踪质量、电荷、动量、能量四种守恒律的状态。每次放置原子或成键都会扰动矩阵。引擎通过 Householder 变换 + QR 迭代（Rust nalgebra 实现）实时计算特征值。

特征值偏离 1.0 的程度决定结构状态：健康（<0.3）→ 警告（0.3-0.6）→ 危险（0.6-0.9）→ 崩解（>0.9）。玩家必须像走钢丝一样维持四种守恒律的平衡。

Mechanic II — Proof Tree

证明树 — 构造即证明

每个操作都是证明树的一个节点。这不是比喻——这是 Brouwer 直觉主义数学的精确实现。在直觉主义中，存在性证明必须给出构造方法，不能只是反证法。你放置的每一个原子、形成的每一个键，都是"这个结构存在"的构造性证明步骤。

玩家可以分叉（探索不同路径）、嫁接（合并子证明）、回溯（撤销错误分支）。证明树深度影响最终得分——最短证明 = 最高分。UI 中每个节点可右键操作，活跃分支高亮，非活跃分支暗淡。

# 证明树操作 ProofTree.fork(node_id) # 从当前节点分叉出新分支 ProofTree.graft(src, dst) # 将子树嫁接到目标节点 ProofTree.backtrack(id) # 回溯：删除节点及后代 # 评分：深度惩罚 score = base * (1.0 + k / depth) # k 越大，短证明的奖励越高
Mechanic III — Undecidability Fog

不可判定迷雾 — 哥德尔可玩化

这是 Intuita 最独特的设计。三种迷雾精确映射可判定性层次，让哥德尔不完备定理成为可玩的博弈：

1 核心 • 确定性驱散
不可判定
3 核心 • 50% 穿透
独立
不可驱散 • 只能绕行

半可判定雾：蓝色薄雾，花 1 核心确定性驱散。映射：停机问题的半可判定性——如果程序停机，你最终会知道。

不可判定雾：紫色漩涡，花 3 核心有 50% 概率穿透。失败时紫色闪电击出。映射：停机问题的不可判定性——你永远无法预先知道结果。

独立雾：纯黑虚无，任何交互都无声反弹。映射：Gödel 独立命题——既不可证真也不可证伪，你只能绕行。

Mechanic V — Categorical Morphisms

范畴论态射系统 — 结构变换的数学

8 种态射类型精确映射范畴论概念，每种有不同的可逆性和组合规则。态射组合不是交换的——f∘g ≠ g∘f——这直接来自范畴论的公理。Surrogate 态射会"吸收"左侧组合（类似范畴中的零态射），Approximation 是"传染性"的（类似拓扑中的开集传递性）。

# 8种态射类型 — 每种映射一个范畴论概念 Transformation # 同构 — 可逆，保持所有结构 Restriction # 单态射 — 注入，信息不丢失 Embedding # 单态射 — 子结构嵌入 Projection # 满态射 — 商结构，信息压缩 Approximation # 一般态射 — 传染性：组合后仍为Approximation Surrogate # 零态射 — 吸收性：Surrogate∘X = Surrogate Composite # 复合态射 — 可逆当且仅当两个分量都可逆 Identity # 恒等态射 — 不改变结构 # 组合规则示例 compose(Transformation, Transformation) = Composite, invertible compose(Restriction, Embedding) = Projection, not invertible compose(Surrogate, X) = Surrogate, # 吸收律 compose(Approximation, X) = Approximation, # 传染性

07 / Value & Significance

价值与意义

商业价值

0
服务器算力成本 — 玩家自带 API Key
  • 极低运营成本。LLM 算力由玩家自行承担（接入自己的 API Key），游戏本身不提供也不存储 API 密钥。核心计算（守恒矩阵特征值）在本地 Rust 引擎完成，无需云端。这意味着我们不需要为 AI 推理付费，运营成本几乎为零。
  • 差异化定位。Foldit 验证了公民科学游戏的可行性，Zachtronics 验证了硬核构造解谜的付费意愿。Intuita 处于两者交集，"数学哲学可玩化"是一个尚未被占据的细分方向。
  • 长线留存设计：37 关卡 + 每日挑战 + 沙盒模式 + 自进化规则系统 + 材料图鉴收集。核心经济系统（验证投资循环 + 核心商店 + 进化点兑换）提供持续游玩动力。
  • 变现模式：基础版免费（Ch1-2）+ 完整版买断 + 教育机构授权。不依赖内购或广告，不卖算力。
