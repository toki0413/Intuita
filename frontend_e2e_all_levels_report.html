发现的问题

1. 路径构建模式关卡无Wyckoff标记 (11关受影响)

C3-L4到C3-L11、C4-L2、C4-L5、C4-L8、C5-L4到C5-L8 等关卡使用"path_build"或特殊模式，不生成Wyckoff标记。 初学者玩家进入后看到空场景，不知道如何开始。建议: 为路径构建模式添加可视化引导(如起点/终点标记)， 或在场景中显示"请使用PATH_BUILD工具点击构建路径"的提示。

2. 目标进度检查不够即时

多个关卡放置了7-8个原子但目标显示0/N。原因: 初学者放置的原子可能不在正确的Wyckoff位置， 或目标需要特定元素(如Na/Cl)而非任意原子。建议: 在放置时显示该位置期望的元素类型， 或在HUD中高亮当前需要放置的元素。

3. BondRenderer pool释放时打印堆栈

删除原子时 _remove_bonds_for_atom 调用 object_pool.gd:57 的 release() 打印调用栈。 这不是崩溃，但说明 pool 的 release 路径有调试日志未清理。建议: 移除或降级为 verbose 级别日志。

4. 上下文指引系统已工作但需进一步优化

ContextualHintSystem 在所有46关中正确触发了工具提示和状态提示。 但对于路径构建模式关卡，提示仍显示"点击蓝色标记放置原子"，实际没有标记。 建议: 根据关卡模式(domain/construction_mode)切换提示内容。

5. C4-L7截图渲染异常

C4-L7 (Mixed Bonds) 的截图中3D视口出现白色噪点/过曝效果。可能是粒子效果或shader在截图时 未完全渲染。建议: 在截图前增加额外帧等待，或检查粒子系统是否干扰渲染。
