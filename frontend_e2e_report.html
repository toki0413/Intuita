KI Known Issues & Limitations

Godot 4 Synthetic Button Event Limitation

push_input(InputEventMouseButton) does NOT trigger input_event on StaticBody3D in Godot 4.6.3, even when hover is confirmed. The viewport's _physics_picking() processes InputEventMouseMotion for hover tracking but does not fire input_event for button events with synthetic input. This is a Godot engine limitation, not a game bug. The HYBRID mode bypasses this by calling _on_input_event() directly after real hover confirmation.

BondRenderer Parent Error

_create_bond in atom_placement_manager.gd:571 throws "Can't add child 'BondRenderer' to 'Bonds', already has a parent" when creating bonds. This occurs because the BondRenderer node is reused without being properly removed from its previous parent. The bond is still created functionally, but the error indicates a cleanup bug in the bond rendering code.

Camera Controller apply_level_scale Bug

apply_level_scale() in camera_controller.gd sets _camera_distance = 0.0 after level load, causing the camera to sit at origin. Workaround: directly set camera position in the test script. This bug should be fixed in the camera controller.
